# リポジトリ名
kojisaki.github.io




## 使い方

英語の質問に対して、英語を発音し答える。

1. 開始するときは、startボタン
2. 英語で回答する
3. 終了するときは、stopボタン

質問の意味が分からないとき・発音が上手く認識できないときは、"let's change the subject"



## 工夫した点

* 日本語だと表記ブレへの対応が大変だと知り、英語を用いた対話にしています。
* 質問をこちらからすることで、何を話すかに戸惑わないように。  
* 人相手に練習するのは恥ずかしい英語も、対話システムを利用するとハードルの低いものに感じられないかと考えました。
